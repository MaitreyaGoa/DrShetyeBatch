📖 How to Start a Live Class

Step 1 – Go to YouTube Studio → Go Live

Step 2 – Copy the YouTube stream link once it's created

Step 3 – Paste the link above, set status to LIVE NOW or SCHEDULED, and click Save

Step 4 – Students visit the Student Stream Page – they'll see the live video player automatically

After class – Set status to OFFLINE and clear the link, or leave it so students can rewatch the recording
